# chaekgalpi

책장 위에 책을 트리 구조로 꽂아두는 시각화 앱.

## 폴더 구조

```
├── README.md
├── index.html
├── package.json
├── tailwind.config.js  (등 설정 파일들)
├── docs/
│   └── SPEC.md
└── src/
    ├── App.tsx
    ├── main.tsx
    ├── types.ts  · theme.ts · index.css
    ├── data/
    │   └── seed.ts
    └── components/
        ├── Shelf.tsx · BookSpine.tsx · Editor.tsx · BookNode.tsx
```

## 개발

```bash
npm install
npm run dev
npm run build
```
